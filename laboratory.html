交换机基础配置实验指南

实验简介

本实验基于Cisco Packet Tracer 7.0及以上环境，重点掌握企业级交换机的初始化配置、VLAN划分、中继链路设置及基础安全策略。实验内容涵盖从物理连接到远程管理的完整流程，适用于网络工程、信息安全等专业学生及企业IT人员。

实验周期：建议4-6课时完成基础配置，进阶内容可扩展至8课时。

B站精选教学视频

  • Cisco交换机基础配置全流程演示（2023新版）

    涵盖Console连接、密码设置、VLAN划分及中继链路配置。

  • VLAN与中继链路实战案例（含PT仿真）

    通过企业级拓扑演示VLAN间通信配置，包含常见错误排查。

  • 交换机安全配置与攻防演示

    讲解端口安全、BPDU防护及DHCP监听等防护策略。

核心配置代码示例

1. 基础系统配置


        enable
        configure terminal
        hostname SW-Office
        enable secret class123
        service password-encryption
        ip default-gateway 192.168.1.254
        line console 0
         password console_pass
         login
        line vty 0 4
         password telnet_pass
         login
        exit
        copy running-config startup-config

2. VLAN与中继配置


        vlan 10
         name Sales
        vlan 20
         name HR
        interface GigabitEthernet0/1
         switchport mode access
         switchport access vlan 10
        interface GigabitEthernet0/24
         switchport mode trunk
         switchport trunk native vlan 99
         switchport trunk allowed vlan 10,20

3. 安全增强配置


        interface GigabitEthernet0/1
         switchport port-security
         switchport port-security maximum 3
         switchport port-security violation restrict
        spanning-tree bpduguard enable
        ip dhcp snooping
        ip dhcp snooping trust interface GigabitEthernet0/24

实验注意事项

  • 初始配置必须通过Console端口，建议使用Putty或SecureCRT工具
  • 管理VLAN建议使用非默认ID（如VLAN 99），并配置ACL限制访问
  • 中继链路两端需保持封装协议（802.1Q）和本征VLAN一致
  • 密码设置遵循复杂度要求，定期更换管理密码
  • 配置变更后执行`show running-config`检查，确保无遗漏

注：本文档基于2023-2024年最新网络技术发展整理，实际操作需结合具体设备型号调整配置参数。

VLAN与中继配置实验指南（2023-2024）

实验简介

本实验基于Cisco Packet Tracer 7.3环境，通过双交换机拓扑掌握VLAN划分、802.1Q中继配置及VLAN间通信原理。实验包含VLAN创建、端口分配、中继协商及本征VLAN配置等核心内容，适合网络工程专业学生及企业网络管理员。

实验周期：建议6-8课时，含进阶配置（如VLAN修剪、DTP模式分析）。

B站精选教学视频

  • VLAN与中继链路配置实战（含PT仿真）

    涵盖VLAN创建、端口分配及802.1Q中继配置全流程。

  • 本征VLAN配置与安全防护

    详解本征VLAN作用及配置注意事项，附攻击演示。

  • DTP协议原理与配置案例

    分析动态中继协议工作机制，演示不同协商模式效果。

核心配置代码示例

1. 基础系统配置


    enable
    configure terminal
    hostname SW-Core
    enable secret class123
    service password-encryption
    ip default-gateway 192.168.1.254
    line console 0
     password console_pass
     login
    line vty 0 4
     password telnet_pass
     login
    exit
    copy running-config startup-config

2. VLAN创建与端口分配


    vlan 10
     name Sales
    vlan 20
     name HR
    interface GigabitEthernet0/1
     switchport mode access
     switchport access vlan 10
    interface GigabitEthernet0/2
     switchport mode access
     switchport access vlan 20

3. 802.1Q中继配置


    interface GigabitEthernet0/24
     switchport mode trunk
     switchport trunk native vlan 99
     switchport trunk allowed vlan 10,20
    spanning-tree portfast trunk

4. 本征VLAN安全增强


    interface GigabitEthernet0/24
     switchport trunk native vlan 999
     switchport nonegotiate
     spanning-tree bpdufilter enable

实验注意事项

  • 中继链路两端必须使用相同的封装类型（802.1Q）
  • 本征VLAN建议使用非默认ID（如VLAN 999）
  • 配置trunk前确保两端端口模式匹配（如dynamic desirable/dynamic auto）
  • 使用`switchport nonegotiate`关闭DTP协商提高安全性
  • 删除VLAN前需先将端口分配到其他VLAN
  • 通过`show interfaces trunk`命令验证中继状态

常见故障处理

故障现象 可能原因 解决方案
PC无法跨VLAN通信 未配置三层路由 添加SVI接口或配置路由器子接口
中继链路协商失败 模式不匹配 设置`switchport mode trunk`强制中继
VLAN流量未通过中继 允许列表错误 使用`switchport trunk allowed vlan`调整
本征VLAN不一致 配置错误 检查`switchport trunk native vlan`

注：实验配置基于Cisco IOS 15.2(2)，不同设备型号可能存在命令差异。

单臂路由器 VLAN 间路由实验指南（2023-2024）

实验简介

本实验基于Cisco Packet Tracer 7.3环境，通过单臂路由器架构实现VLAN间通信。重点掌握子接口配置、802.1Q中继及路由策略，适合网络工程专业学生及企业网络管理员。

实验周期：建议8-10课时，含进阶配置（如ACL访问控制、DHCP中继）。

实验目标

  • 构建网络并配置设备基本设置
  • 创建VLAN并分配交换机端口
  • 配置802.1Q中继链路
  • 实现单臂路由器VLAN间路由
  • 验证端到端通信

地址分配表

设备 接口 IP 地址 子网掩码 默认网关
R1 G0/0/1.3 192.168.3.1 255.255.255.0 —
S1 VLAN 3 192.168.3.11 255.255.255.0 192.168.3.1
PC-A NIC 192.168.3.3 255.255.255.0 192.168.3.1

核心配置代码示例

1. 路由器子接口配置


        enable
        configure terminal
        interface GigabitEthernet0/0/1.3
         description Management Network
         encapsulation dot1q 3
         ip address 192.168.3.1 255.255.255.0
        interface GigabitEthernet0/0/1.4
         description Operations Network
         encapsulation dot1q 4
         ip address 192.168.4.1 255.255.255.0

2. 交换机中继配置


        interface FastEthernet0/1
         switchport mode trunk
         switchport trunk native vlan 8
         switchport trunk allowed vlan 3,4,8

3. VLAN间路由验证


        show ip route
        show interfaces trunk
        ping 192.168.4.3

实验注意事项

  • 子接口封装类型必须与VLAN ID匹配
  • 中继链路两端本征VLAN需一致
  • 启用路由器接口的IP路由功能
  • 验证ARP表确保地址解析正常
  • 使用`debug ip packet`跟踪路由过程

实施以太通道实验指南（2023-2024）

实验简介

本实验基于Cisco Packet Tracer 7.3环境，通过双交换机配置链路聚合（LACP）实现带宽扩展与冗余。重点掌握以太通道配置、中继链路优化及聚合状态验证。

实验周期：建议6-8课时，含进阶配置（如PAgP协议分析、负载均衡策略）。

实验目标

  • 构建网络并配置设备基本设置
  • 创建VLAN并分配交换机端口
  • 配置802.1Q中继链路
  • 实现LACP以太通道
  • 验证链路聚合状态

地址分配表

设备 接口 IP 地址 子网掩码
S1 VLAN 10 192.168.10.11 255.255.255.0
PC-A NIC 192.168.20.3 255.255.255.0

核心配置代码示例

1. 交换机基本配置


        enable
        configure terminal
        hostname SW1
        enable secret class123
        service password-encryption
        ip default-gateway 192.168.1.254

2. 以太通道配置


        interface range FastEthernet0/1-2
         switchport mode trunk
         switchport trunk native vlan 1000
         switchport trunk allowed vlan 10,20,1000
         channel-group 1 mode active

3. 验证命令


        show etherchannel summary
        show interfaces trunk
        show ip interface brief

实验注意事项

  • 物理端口必须属于同一模块且速率/双工一致
  • 确保两端交换机的LACP模式匹配
  • 本征VLAN需与中继配置一致
  • 使用`channel-group`命令前先配置中继参数
  • 验证`Port-channel`接口状态

VLAN间路由故障排除实验指南（2023-2024）

实验简介

本实验基于Cisco Packet Tracer 7.3环境，通过预配置故障场景掌握VLAN间路由排错方法。重点学习路由表分析、中继配置检查及常见错误修复。

实验周期：建议8-10课时，含进阶内容（如ACL调试、DHCP中继问题处理）。

实验目标

  • 评估网络初始状态
  • 收集配置信息并分析故障
  • 修复VLAN间路由配置错误
  • 验证端到端连通性

地址分配表

设备 接口 IP 地址 子网掩码 默认网关
R1 G0/0/1.3 10.3.0.1 255.255.255.0 —
PC-A NIC 10.4.0.50 255.255.255.0 10.4.0.1

核心修复命令示例

1. 路由器配置修复


    interface g0/0/1.8
     encapsulation dot1q 8 native
     exit

2. 交换机配置修复


    default interface f0/5
    interface f0/5
     switchport mode trunk
     switchport trunk allowed vlan 3,4,8,13

3. 验证命令


    show running-config
    show ip route
    ping 10.13.0.50

实验注意事项

  • 检查VLAN是否存在及端口分配
  • 确认中继链路本征VLAN一致性
  • 验证子接口封装与VLAN ID匹配
  • 使用`debug`命令跟踪数据包
  • 注意ACL对流量的隐含拒绝
