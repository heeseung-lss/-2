实战项目连接
  • 校园网IPv6升级项目：在校园网环境中，可实施IPv6升级项目。例如，北京大学在校园网IPv6升级过程中，首先对校园内网络设备进行IPv6双栈配置，通过在核心交换机和路由器上启用IPv6协议，配置全球单播地址和链路本地地址等。具体配置如在华为路由器上：
    interface GigabitEthernet0/0 ipv6 enable ipv6 address 2001:db8::1/64
  • 企业数据中心IPv6改造：以某大型企业数据中心为例，为满足业务全球化和网络扩展性需求，进行IPv6改造。先规划全球单播地址（2000::/3）用于公网通信，配置链路本地地址（FE80::/10）用于内部链路通信。同时，利用双栈技术在现有网络设备上同时运行IPv4和IPv6协议，在Cisco设备上配置如下：
    interface GigabitEthernet0/0 ipv6 enable ipv6 address 2001:db8::1/64 ip address 192.168.1.1 255.255.255.0
